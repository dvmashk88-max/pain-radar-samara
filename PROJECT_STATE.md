## Известные проблемы

1. SQLite на Railway может потеряться при пересоздании контейнера.
   Для тестов нормально.
   Для боевого накопления за месяц лучше потом перейти на:
   - Railway PostgreSQL
   - или Supabase

2. Telegram public иногда даёт мусорные новости не по бизнес-болям.
   Нужно улучшить фильтр:
   - учитывать только тексты, где есть город/регион;
   - отсеивать криминал, ДТП, политику, военные новости;
   - фокус на бизнес-услуги, сервис, клиники, автосервисы, ЖКХ.

3. DuckDuckGo часть сигналов берёт из заголовков, а не из реальных отзывов.
   Это пока нормально для MVP, но нужно улучшать качество.

4. VC.ru и Habr Q&A пока заглушки.

5. Нет PostgreSQL.
6. Нет публичного канала.
7. Нет подписки/монетизации.
8. Нет экспорта CSV.
